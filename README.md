VhyeCvarHelper Addon
====================

Description:
------------
VhyeCvarHelper is a replacement for WeakAuras functionality; used to configure and maintain persistent cvar values for World of Warcraft.
- Update cvar values when you login or reload
- Provides a GUI allowing you to manage, maintain and import/export cvars
- Provides an option to reset all CVars to their default values

Installation:
-------------
1. Download or copy the folder `VhyeCvarHelper`.
2. Place it in your WoW AddOns directory:
   World of Warcraft/_retail_/Interface/AddOns/
3. Restart WoW or type `/reload`.

Usage:
------
- Type `/vhye` in chat to open the GUI.
- Use the search box to filter CVars.
- Select a CVar from the dropdown, enter a value, and click "Add/Update".
- Click "Reset All to Defaults" to restore all CVars.
- Use Export/Import to share or load CVar sets.

Saved Variables:
----------------
Your settings are saved in `VhyeCvarHelperDB` and persist between sessions.
